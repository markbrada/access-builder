# Exec Plan: v0.1 Ramp + Platform Designer

## Goal
Ship a GitHub Pages app that lets a user place/edit ramps and platforms in 2D, preview in 3D, and export a dimensioned PDF.

## Milestones
M1 Project scaffold
- Vite + React + TS, linting, basic routing optional
- GitHub Pages deploy workflow

M2 UI shell
- Top bar, left toolbox (Ramp, Platform), centre canvas with grid, right inspector

M3 2D editor MVP
- Place/select/move/rotate
- Resize handles
- Snap-to-grid
- Ramp arrow
- Derived ratio + gradient

M4 3D preview (view-only)
- Toggle to 3D
- Render ramp and platform meshes
- Orbit controls

M5 PDF export
- Export current view to PDF with title block and dimensions

## Definition of Done
- All milestones complete
- Manual test checklist passes
